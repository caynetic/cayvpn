Troubleshooting Guide
Common Issues & Solutions
  • Local DNS fails: AdGuard Home may not be running or not listening on 10.8.0.1:53
  • DNS service not found: Check if AdGuard Home is installed and running
  • AdGuard service inactive: Start service with sudo systemctl start AdGuardHome
  • WireGuard issues: Ensure wg0 interface is up and routing is correct
Quick Commands
  • sudo systemctl status AdGuardHome - Check service status
  • sudo systemctl restart AdGuardHome - Restart AdGuard
  • ip addr show wg0 - Check WireGuard interface
  • wg show wg0 - Show WireGuard peers and status
  • iptables -L | grep 53 - Check DNS port forwarding
